VidCaster

VidCaster asked me to create a theme for their content management system focused on video. While the project only called for me to supply the design, I worked closely with VidCaster's developer to turn my Photoshop design into working HTML / CSS templates. Together, we developed page load techniques and used CSS3 in a way to allow for graceful degradation.

Gig Salad

Gig Salad is a leading site for booking live entertainment. While the website reflects a strong brand and intuitive interface, their email notifications had traditionally been fairly basic.

I was asked to create a few HTML email templates for the in-house team to repurpose and use as needed. I designed a few branded templates and wrote mobile friendly code that display well even before images downloaded. Each template was thoroughly tested and shipped with lots of code comments.

4ormat

I designed and coded a theme for 4ormat, which allowed its users to upload a wide variety of content and change the appearance of their profile. The theme is light on the UI and interaction effects, allowing the content to do the talking.

When it came time to code, I learned 4ormat's template language to nest variables directly into the CSS, allowing users to change the theme's appearance. When all HTML, CSS, and JavaScript validated, the theme shipped and became part of 4ormat.com.
